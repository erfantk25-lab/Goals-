# AI Goal Planner System (Brian Tracy 12-Step Generator)

This is a full production-grade MLOps + AI system.

## Project Structure

- `api/`: FastAPI backend
- `ml/`: ML training and inference pipelines
- `db/`: PostgreSQL models and Alembic migrations
- `prompts/`: Versioned LLM prompts
- `docker/`: Docker and Compose configuration files
- `infra/`: Deployment scripts
- `scripts/`: Linux/bash utilities
- `tests/`: Pytest suite
- `docs/`: Architecture documentation
- `core/`: Core configurations and logging

## Setup

1. Create a virtual environment: `python -m venv venv`
2. Activate the virtual environment
3. Install dependencies: `pip install -r requirements.txt`
